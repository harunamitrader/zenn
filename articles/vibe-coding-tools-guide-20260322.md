## コード作業寄り

### Codex
OpenAIのコーディングエージェントで、CLI・Web・アプリ・IDEなど入口が多いです。  
費用: 一部無料（ちょっとしか使えない） / 20$〜

### Claude Code
Anthropicのターミナル中心ツールで、端末の中でAIと一緒に作業する感じがかなり強いです。  
費用: 有料 / 20$〜

### Cursor
AI前提で作られたコードエディタで、自然文での編集やエージェント機能までまとめて使えます。  
費用: 一部無料（ちょっとしか使えない） / 20$〜

### Windsurf
`Cascade` を中心に、会話しながら編集と実行を進めるAI開発環境です。  
費用: 一部無料（ちょっとしか使えない） / 15$〜

### GitHub Copilot
補完だけでなく、GitHub上でagentっぽく使う流れまで入ってきた定番ツールです。  
費用: 一部無料（ちょっとしか使えない） / 10$〜

### Gemini CLI
Google公式のCLI型AIツールで、ターミナルからGeminiを使いたい人向けです。  
費用: 一部無料（無料で複数のアプリを開発できるぐらい使える）

### Antigravity
Googleのagent-firstな開発プラットフォームで、複数エージェントに仕事を振る感じが強いです。  
費用: 一部無料（無料で複数のアプリを開発できるぐらい使える）

### Jules
Googleの非同期コーディングエージェントで、クラウド側で進めてPRまで持っていくタイプです。  
費用: 一部無料（無料で複数のアプリを開発できるぐらい使える）

### Amazon Q Developer
AWS色は強いですが、CLIやエージェント的な使い方もできる企業向けの有力候補です。  
費用: 一部無料（無料で複数のアプリを開発できるぐらい使える）

### Aider
ターミナル派に根強い人気がある、オープンソースのAIペアプロツールです。  
費用: ツール自体は無料（API代を従量課金）

### Roo Code
VS Code拡張として無料で使い始めやすく、必要に応じてCloud Agentsも足せるツールです。  
費用: ツール自体は無料（API代を従量課金）

## アプリ開発寄り

### Replit Agent
「作りたいものを言うと形にしていく」方向が強く、非エンジニアでも入りやすいです。  
費用: 一部無料（ちょっとしか使えない） / 20$〜

### v0
Vercelの生成UI・アプリ作成ツールで、Web画面やたたき台づくりがかなり得意です。  
費用: 一部無料（ちょっとしか使えない） / 20$〜

### Lovable
会話ベースでフルスタックWebアプリを作る方向が強く、コードを書かずに進めたい人向けです。  
費用: 一部無料（ちょっとしか使えない） / 25$〜

### Bolt
会話しながらアプリやサイトを作っていくツールで、vibe coding感がかなり前に出ています。  
費用: 一部無料（無料で複数のアプリを開発できるぐらい使える）

### Google AI Studio
Geminiを試したり、Build modeで小さなアプリを作ったりしやすいGoogle公式の入口です。  
費用: 完全無料

## まとめ

私みたいな非エンジニアの初心者からすると、まず大事なのは「どれが最強か」を決めることではなく、**自分が触りやすい入口を見つけること** なんだと思います。

コードを書く人向けのツールもあれば、会話しながらアプリを作るのが得意なツールもあります。  
しかも、無料でかなり試せるものもあれば、最初から有料前提のものもあります。

なので最初は、

- コード作業寄りを触ってみたいなら `Codex`、`Cursor`、`Windsurf`、`Claude Code`
- アプリを作るところから始めたいなら `Replit Agent`、`Lovable`、`Bolt`、`v0`
- まず無料でいろいろ試したいなら `Google AI Studio` や `Gemini CLI`

みたいに、**自分のやりたいことに近いものを1つ選ぶ** くらいで十分そうです。
